# pongGame
Ho ricreato il gioco pong utilizzando le librerie LibGDX

Non c'è molto da dire è un semplice progettino che ho creato per passare il tempo. 
Il codice non è molto commentato ma è abbastanza intuitivo, penso che comunque aggiungerò e sistemerò varie cose nel tempo.

Per la build utilizzate una vecchia versione java (consiglio 8.0) se no potrebbe sare problemi.

Per l'esecuzione una qualsiasi versione di java dovrebbe andare bene. 

Creato con Netbeans 8.2 / 12.6.

8.2 -- Gradle plugin.
